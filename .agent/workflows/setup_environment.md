---
description: Sets up the development environment for the Renan Crociari Portfolio project.
---

1. **Verify Node.js Installation**
   Ensure Node.js is installed.
   ```bash
   node -v
   ```

2. **Verify Git Installation**
   Ensure Git is installed.
   ```bash
   git --version
   ```

3. **Install Dependencies**
   Install all project dependencies defined in package.json.
   // turbo
   ```bash
   npm install
   ```

4. **Verify Build**
   Run a production build to ensure everything is set up correctly.
   ```bash
   npm run build
   ```

5. **Start Development Server**
   Start the local development server.
   ```bash
   npm start
   ```
